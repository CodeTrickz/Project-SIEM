### omschrijving project
Project Claritas is een beveiligingsproject dat gericht is op het verbeteren van de infrastructuur van gemeenten op het gebied van beveiliging. Als onderdeel van dit project moet ons team een Security Information and Event Management (SIEM) systeem ontwerpen. Deze SIEM zal alle informatie die de gemeentelijke infrastructuur uitzendt analyseren.  Het SIEM systeem zal in staat zijn om verdachte activiteiten te detecteren, potentiÃ«le bedreigingen te identificeren en proactief te reageren op beveiligingsincidenten. Ons doel is om de veiligheid van de gemeentelijke systemen te versterken en een extra laag te bouwen tegen cyberdreigingen.
